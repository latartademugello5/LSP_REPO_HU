# IntegerSet Documentation

## Project Overview
This project implements a custom IntegerSet class that provides various set operations for integer collections. The implementation ensures no duplicate elements and offers methods for common set manipulations.

## Documentation Contents
This documentation provides detailed information about the IntegerSet class methods, including:
- Constructor details
- Method descriptions
- Parameter explanations
- Return value specifications
- Exception handling

## Key Methods
- `add(int item)`: Adds a unique integer to the set
- `remove(int item)`: Removes a specific integer from the set
- `union(IntegerSet intSetb)`: Combines two sets
- `intersect(IntegerSet intSetb)`: Finds common elements between sets
- `diff(IntegerSet intSetb)`: Performs set difference operation
- `contains(int value)`: Checks for element presence
- `largest()`: Finds the maximum value in the set
- `smallest()`: Finds the minimum value in the set

## Usage Notes
- The set does not allow duplicate elements
- Methods throw appropriate exceptions for edge cases
- All operations maintain the set's unique element property
